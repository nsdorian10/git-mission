## Description
A simple command-line authentication system using bcrypt for secure password hashing.

## Features
- Password hashing with bcrypt
- User registration & login
- Input validation
- File-based storage

## Installation & Usage
1. Install: `pip install bcrypt`
2. Run: `python auth.py`

## Files
- `auth.py` - Main code
- `users.txt` - User database
- `requirements.txt` - Dependencies
